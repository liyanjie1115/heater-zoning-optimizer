Heater Zoning Optimizer Desktop
Version: 1.0.0

1. Start
- Double-click `heater-zoning-optimizer.exe`

2. Recommended first run
- Select sample data
- Click Run Analysis
- Review the recommendation summary first
- Then review metric details and difference summary

3. Input data
- CSV and Excel are supported
- Required columns:
  - `distance_mm`
  - `temperature_c`

4. Sample file
- `sample-data/sample_profile.csv`

5. Output
- Excel report is generated after analysis
- PDF summary can also be exported from the desktop app

6. Notes
- Recommended for Windows 10 / 11
- If Windows shows a first-run warning, choose to continue
